# AI Orchestrator Suite v3 — Multi-IA com Decisão Manual/Automática

## Visão Geral
A suíte orquestra múltiplos agentes de IA (Codex, Claude, Gemini) para entregar funcionalidades completas com rastreabilidade, failover automático e controle de decisão (manual ou autônomo). Este diretório reúne documentação operacional, especificações e histórico de decisões.

## Conteúdo
- `SPEC.md`: visão funcional e técnica do orquestrador.
- `ADR-0001-inicial.md`: decisão arquitetural inicial.
- `operating-manual.md`: comandos e fluxos de trabalho.
- `ANALISE_PROJETO.md`: catálogo de reuso/decisões que deve ser atualizado a cada evolução, manutenção ou remoção.

## Como Navegar
1. Consulte `SPEC.md` para entender módulos e interfaces.
2. Leia as ADRs para conhecer justificativas de arquitetura.
3. Utilize o manual operacional para execução diária (init, run, failover).
